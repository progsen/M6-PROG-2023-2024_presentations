## Fetch uitleg

- `Fetch` is `async`
    - Je laat je code uitvoeren
    - En koppelt een `Als je klaar bent` stuk code
        - `.then()`
        - tussen de `()` van `then` komt een function
> Deze manier van werken heet `promises` in javascript
